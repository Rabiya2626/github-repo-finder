# 🚀 GitHub Repo Finder

A modern React application to search GitHub repositories in real-time.

## ✨ Features
- Debounced search (optimized API calls)
- Favorites system (localStorage)
- Clean UI with Tailwind CSS
- Routing with React Router

## 🛠 Tech Stack
- React
- Vite
- Tailwind CSS
- GitHub API
